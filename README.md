# Customer Churn Analysis

## 🎯 Objective
This project analyzes telecom customer data to identify the key drivers of churn and build a predictive model to reduce customer attrition.

## 📊 Key Insights
* **Contract Type:** Customers on month-to-month contracts are 3x more likely to churn than those on two-year plans.
* **Tech Support:** Absence of online tech support is the #1 predictor of early churn.
* **Fiber Optic:** Surprisingly, Fiber Optic users have a higher churn rate, suggesting potential service quality issues.

## 🛠️ Tech Stack
* **Language:** Python (Anaconda Environment)
* **Libraries:** Pandas (Data Cleaning), Seaborn/Matplotlib (Visualization), Scikit-Learn (Modeling)
